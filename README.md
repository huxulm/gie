## 安装 & 使用
```shell
go install github.com/huxulm/gie/cmd
```
## 使用
所有参数、子命令都直接传递到 `git` 中，除了 `git commit` 子命令会额外处理。
```
# use gie instead of git
gie commit -m ":art: welcome use gie!"
```
